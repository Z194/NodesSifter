# 程序目的：
在日常使用v2rayN过程中发现，同IP地址节点即使端口不同，它们的速度是一样的，那些有几个甚至几十个端口的，只要有一个端口失效，其他同IP节点都会失效，所以很有必要将IP节点筛选到只留一个，因为这将节省很多测速资源

## 支持节点：
      trojan、ss、vmess

## 使用方法：
      过滤节点：
            1.将节点URL地址粘贴到‘nodes_pool.txt’
            2.运行 NodesSifter.py
            3.完成，结果保存在‘finish_nodes_pool.txt’
